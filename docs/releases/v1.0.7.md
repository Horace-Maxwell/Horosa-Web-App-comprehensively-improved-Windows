# 星阙 Horosa Windows 桌面版 `v1.0.7`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.7.exe`
- **如果你已经安装 `v1.0.6` 或更早版本**：建议升级到 `v1.0.7`
- GitHub Release 只保留离线安装器，不提供 portable zip 或源码快照式资产

## 三步安装

1. 下载 `Horosa-Setup-1.0.7.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `Horosa`（应用窗口标题仍显示“星阙”）

## 离线说明

- 安装包已内置 Electron、Java、Python、前后端资源
- 安装和本地功能可离线运行
- 普通用户机器不需要预装开发环境、浏览器、Java 或 Python
- 若要升级，请重新下载最新完整安装器覆盖安装
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次热修内容

### 1. 关闭后彻底退出

- 点击右上角 `X` 会完整退出桌面应用，不再把 Electron 主进程残留在后台
- 退出时会同步停止本地 Java 后端、Python 图表服务和相关子进程
- AppCDS 动态导出增加超时保护，不再因为退出阶段卡住而留下僵尸实例

### 2. 再次双击可重新打开

- 正常关闭后，再次双击 `Horosa.lnk` 或 `Horosa.exe` 会重新冷启动
- 不再出现“上一次关掉后后台没退干净，下一次双击完全没反应”的情况

### 3. 僵尸态与无窗实例自救修正

- 如果已有实例还活着但窗口已损坏，再次双击会直接重建主窗口
- 自救过程不再误触发完整退出链
- 应用正常运行时重复双击仍然只会把已有窗口拉到最前，不会新开第二个主实例

### 4. 发布契约保持稳定

- Release 只提供 `Horosa-Setup-*.exe`
- 快捷方式继续为 `Horosa`
- 窗口标题继续显示“星阙”
- 不再依赖 `latest.yml`、`.blockmap` 或应用内自动更新元数据

## 适用对象

- **普通用户**：直接下载 `.exe` 完整离线安装器
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支
