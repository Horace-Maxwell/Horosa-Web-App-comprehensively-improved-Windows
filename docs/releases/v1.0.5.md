# 星阙 Horosa Windows 桌面版 `v1.0.5`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.5.exe`
- **如果你已经安装 `v1.0.4`**：建议升级到 `v1.0.5`
- GitHub Release 只保留离线安装器，不提供 portable zip 或源码快照式资产

## 三步安装

1. 下载 `Horosa-Setup-1.0.5.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `Horosa`（应用窗口标题仍显示“星阙”）

## 离线说明

- 安装包已内置 Electron、Java、Python、前后端资源
- 安装和本地功能可离线运行
- 普通用户机器不需要预装开发环境、浏览器、Java 或 Python
- 若要升级，请重新下载最新完整安装器覆盖安装
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次热修内容

### 1. 双击后立即见窗

- 双击 `Horosa.exe` 或桌面快捷方式后，会先显示“星阙启动中”原生窗口
- 本地 Java / Python 服务继续在后台启动，不再要求用户等待首屏完全 ready 才见窗
- 若启动失败，仍保留启动窗用于展示失败状态、日志入口和重试入口

### 2. 单实例前台拉起修正

- 再次双击应用时，会把已有窗口恢复并拉到最前
- 不再出现进程已在运行但用户体感像“没打开”的情况

### 3. 安装器与快捷方式修复

- 安装器会清理旧入口并重建开始菜单 / 桌面快捷方式
- 安装后会验证快捷方式目标是否真的指向 `Horosa.exe`
- 继续兼容 OneDrive 桌面等 Windows Shell 真实桌面目录场景

### 4. 发布契约继续收口

- Release 只提供 `Horosa-Setup-*.exe`
- 安装版继续使用“重新下载安装器覆盖安装”的升级方式
- 不再依赖 `latest.yml`、`.blockmap` 或应用内自动更新元数据

## 适用对象

- **普通用户**：直接下载 `.exe` 完整离线安装器
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支
