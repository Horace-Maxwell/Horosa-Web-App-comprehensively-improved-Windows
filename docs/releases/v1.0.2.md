# 星阙 Horosa Windows 桌面版 `v1.0.2`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.2.exe`
- **如果你已经下载过 `v1.0.1`**：请改下 `v1.0.2`，这一版继续修复图标和快捷方式问题
- **不要手动下载**：`latest.yml`、`Horosa-Setup-1.0.2.exe.blockmap`
  - 这两个文件是给应用内自动更新使用的

## 三步安装

1. 下载 `Horosa-Setup-1.0.2.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `星阙`

## 离线说明

- 本安装包已经内置 Electron、Java、Python、前后端资源
- 安装后本地功能可离线运行
- 只有“检查更新”需要联网
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次热修内容

### 1. 应用图标修复

- 修复 `Horosa.exe` 仍显示旧默认 Electron 图标的问题
- 继续统一安装器、卸载器、应用窗口、任务栏和快捷方式图标为高清白底黑字 `星阙`
- 改为两段式打包链，确保 EXE 图标在正式安装包中稳定写入

### 2. 快捷方式修复

- 修复开始菜单和桌面快捷方式可能残留旧 `Horosa Desktop.lnk` 的问题
- 修复 `星阙.lnk` 可启动但元数据损坏的情况
- 安装后会显式清理旧入口并重建新的 `星阙.lnk`

### 3. 既有桌面热修继续保留

- 中文维护页：已安装检测、替换 / 修复 / 取消
- 默认窗口比例、缩放、快捷键、窗口恢复逻辑
- 顶栏抽屉、易与三式、组合盘 / 时空中点盘等白屏修复
- 全部主 tab 底部越界修复
- 风水 tab 在桌面壳中的本地资源加载修复

## 适用对象

- **普通用户**：直接下载 `.exe` 安装器
- **已安装 `v1.0.1` 的用户**：建议升级到 `v1.0.2`
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支中
