# 星阙 Horosa Windows 桌面版 `v1.0.8`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.8.exe`
- **如果你已经安装 `v1.0.7` 或更早版本**：建议升级到 `v1.0.8`
- GitHub Release 只保留离线安装器，不提供 portable zip 或源码快照式资产

## 三步安装

1. 下载 `Horosa-Setup-1.0.8.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `Horosa`（应用窗口标题仍显示“星阙”）

## 离线说明

- 安装包已内置 Electron、Java、Python、前后端资源
- 安装和本地功能可离线运行
- 普通用户机器不需要预装开发环境、浏览器、Java 或 Python
- 若要升级，请重新下载最新完整安装器覆盖安装
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次热修内容

### 1. 节气盘标签显示修复

- 修复节气盘各个 `XX星盘` 左上角出现 `undefined, 整宫制` 的问题
- 现在会按当前参数显示 `回归黄道 / 恒星黄道` 和对应分宫制中文名称
- 左上角标签与右侧信息面板的参数显示保持一致

### 2. 节气盘黄道与分宫制切换生效

- 修复节气盘缓存只按节气名复用导致的旧图残留
- 切换上方黄道和分宫制后，`春分 / 夏至 / 秋分 / 冬至` 星盘与 3D 盘都会按当前设置重新取图
- 从总览页切回具体节气图时，也不会再误复用旧参数图盘

### 3. 关系盘底部裁切修复

- 修复比较盘、组合盘、影响盘、时空中点盘、马克斯盘底部黄道圈和星座文字被窗口下沿挡住的问题
- 关系盘现在会按真实可用视口高度自适应缩放，并在底部保留安全间距
- 在正常窗口和偏矮窗口下，关系盘各 Tab 及其二级 Tab 都会优先缩小星盘，而不是继续被底部裁切

### 4. 发布契约保持稳定

- Release 只提供 `Horosa-Setup-*.exe`
- 快捷方式继续为 `Horosa`
- 窗口标题继续显示“星阙”
- 不再依赖 `latest.yml`、`.blockmap` 或应用内自动更新元数据

## 适用对象

- **普通用户**：直接下载 `.exe` 完整离线安装器
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支
