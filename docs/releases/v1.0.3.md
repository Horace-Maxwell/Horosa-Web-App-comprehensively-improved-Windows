# 星阙 Horosa Windows 桌面版 `v1.0.3`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.3.exe`
- **如果你已经安装 `v1.0.2`**：建议升级到 `v1.0.3`
- **不要手动下载**：`latest.yml`、`Horosa-Setup-1.0.3.exe.blockmap`
  - 这两个文件给应用内自动更新使用，不是手动安装入口

## 三步安装

1. 下载 `Horosa-Setup-1.0.3.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `星阙`

## 离线说明

- 安装包已内置 Electron、Java、Python、前后端资源
- 安装和本地功能可离线运行
- 普通用户机器不需要预装开发环境、浏览器、Java 或 Python
- 只有“检查更新”与“下载更新”需要网络
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次热修内容

### 1. 安装版应用内更新修复

- 修复安装版启动后检查更新报 `app-update.yml` 缺失的问题
- 更新器改为显式使用 GitHub Releases 发布源配置，不再依赖本地 `resources/app-update.yml`
- 保留现有 `latest.yml + blockmap + 重启安装` 更新链路

### 2. 离线体验兜底

- 后台自动检查更新在断网或网络异常时改为静默失败
- 不会因为更新源异常影响首次启动或本地离线使用
- 手动点击“检查更新”时仍会给出明确中文结果

### 3. 快捷方式与安装器稳定性

- 桌面快捷方式继续统一为 `星阙.lnk`
- 重复运行安装器或替换安装时，桌面快捷方式按 Windows Shell 实际桌面目录稳定重建
- 兼容 OneDrive 桌面场景，不再用硬编码桌面路径判断是否安装成功

### 4. 既有桌面版修复继续保留

- 真正的 Electron 桌面窗口与完整离线安装包
- 中文维护页：已安装检测、替换 / 修复 / 取消
- 默认窗口 85%、缩放、快捷键、窗口恢复逻辑
- 顶栏抽屉、易与三式、组合盘 / 时空中点盘等白屏修复
- 主 tab 底部越界修复
- 风水 tab 本地资源加载修复

## 适用对象

- **普通用户**：直接下载 `.exe` 完整离线安装器
- **已安装旧版用户**：可通过完整安装器重装，或在联网时使用应用内更新
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支

> 提示：此版本之后已有 `v1.0.4`，普通用户建议下载安装新版。
