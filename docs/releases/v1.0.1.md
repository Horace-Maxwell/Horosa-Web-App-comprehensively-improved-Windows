# 星阙 Horosa Windows 桌面版 `v1.0.1`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.1.exe`
- **如果你已经下载过 `v1.0.0`**：请改下 `v1.0.1`，`v1.0.0` 是旧构建包
- **不要手动下载**：`latest.yml`、`Horosa-Setup-1.0.1.exe.blockmap`
  - 这两个文件是给应用内自动更新使用的

## 三步安装

1. 下载 `Horosa-Setup-1.0.1.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `星阙`

## 离线说明

- 本安装包已经内置 Electron、Java、Python、前后端资源
- 安装后本地功能可离线运行
- 只有“检查更新”需要联网
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据
