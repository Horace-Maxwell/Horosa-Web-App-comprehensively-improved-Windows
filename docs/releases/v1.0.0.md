# 星阙 Horosa Windows 桌面版 `v1.0.0`

## 给谁下载

- **普通 Windows 10/11 x64 用户**：下载 `Horosa-Setup-1.0.0.exe`
- **不要手动下载**：`latest.yml`、`Horosa-Setup-1.0.0.exe.blockmap`
  - 这两个文件是给应用内自动更新使用的

## 三步安装

1. 下载 `Horosa-Setup-1.0.0.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `星阙`

## 离线说明

- 本安装包已经内置 Electron、Java、Python、前后端资源
- 安装后本地功能可离线运行
- 只有“检查更新”需要联网
- 默认安装到 `%LocalAppData%\Programs\Horosa`
- 用户数据保存在 `%LocalAppData%\HorosaDesktop`
- 升级、替换、修复默认保留用户数据

## 本次新功能与修复

### 1. 正式桌面化

- 从浏览器 `--app` 模式升级为真正的 Electron 桌面应用
- 使用独立软件窗口运行，不再依赖系统浏览器外壳
- 统一使用 `horosa_setup.ico` 作为应用、安装器、卸载器和快捷方式图标

### 2. 完整离线安装

- 安装器内置 Electron、Java 17、Python 3.11、前端静态资源与后端运行所需内容
- 尽量降低对目标机器预装环境的要求
- 安装包下载后即可离线安装和本地运行

### 3. 应用内更新

- 接入 GitHub Releases + `latest.yml` + blockmap 更新链路
- 应用内支持检查更新、下载更新、重启安装
- 更新过程中默认保留用户数据

### 4. 安装器成熟化

- 增加“已安装检测”
- 若检测到已安装版本，支持 `替换 / 修复 / 取消`
- 维护页改为更成熟的中文版向导式交互

### 5. 窗口与启动体验

- 默认窗口按工作区比例打开，并允许用户自由缩放
- 支持窗口缩放快捷键
- 优化了启动体验，优先显示桌面壳，再异步拉起本地服务

### 6. 稳定性修复

- 修复顶栏抽屉与多个主模块的白屏问题
- 修复 `易与三式`、组合盘、时空中点盘相关问题
- 修复全部主 tab 底部越界问题
- 修复 `风水` tab 在桌面壳中的本地资源加载问题

## 适用对象

- **普通用户**：直接下载 `.exe` 安装器
- **已安装旧版本的用户**：可直接运行新安装器，或在应用内检查更新
- **开发者**：源码、脚本与桌面工程仍保留在 `main` 分支中
