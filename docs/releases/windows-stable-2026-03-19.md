# Horosa Windows Stable `windows-stable-2026-03-19`

## 给谁下载

- **需要非 App 浏览器壳稳定版的用户**：下载 `HorosaPortableWindows-windows-stable-2026-03-19.zip`
- **普通用户**：优先下载安装版 `Horosa-Setup-1.0.4.exe`

## 使用方式

1. 下载稳定版压缩包并解压
2. 双击 `START_HERE.bat`
3. 等待本地服务启动并进入星阙窗口

## 本次修复

- 默认窗口改为最大化启动
- 页面内容缩放保持原有稳定版比例
- 保留本地服务复用、日志与启动诊断链路

## 说明

- 这是非 App 版，不是 Electron 安装版
- 如果你需要完整离线安装器、应用内更新和桌面快捷方式，请使用 `v1.0.4`
