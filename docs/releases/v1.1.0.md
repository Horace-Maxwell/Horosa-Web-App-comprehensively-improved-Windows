# Horosa v1.1.0

本次正式版聚焦于把此前已经在本机安装版验证通过的修复，完整收口到 GitHub `main` 与 Windows 安装器。

## 重点修复

- 修复遁甲盘主盘缩放，按窗口底边限高，超宽或偏矮窗口下不再被底部裁切
- 修复遁甲盘右侧工具栏布局，底部内容可滚动查看
- 修复节气盘黄道 / 分宫制显示与切换重算
- 修复主限法盘、法达星限空白与 `param error` 相关前端回归
- 恢复主限法 / 法达星限中的后天宫位与主宰宫标注
- 修复六壬与三式合一中的起课、旬法、遁干、三传等规则链问题
- 修复安装版 Python 图表服务与新版 `swisseph.azalt(...)` 的兼容问题
- 修复正式安装器快捷方式创建，使桌面和开始菜单入口指向正式 `Horosa.exe`
- 修复正常关闭后残留进程导致无法再次开启的问题
- 修复 planned shutdown 期间误报 `Runtime error` / `Python chart service exited unexpectedly`

## 发布说明

- Latest Release 只保留 `Horosa-Setup-1.1.0.exe`
- 本版本继续使用完整离线安装器交付 Electron、Java、Python 与前后端资源
- 发布后已按 GitHub Release 下载路径再次回测本机安装链路
