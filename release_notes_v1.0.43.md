# 星阙 v1.0.43 发布说明

发布日期：2026-03-16

## 本次升级重点

- 全量完成最新版网页端、桌面端、轻量在线安装包、离线完整版安装包、便携发布包同步更新
- 完成顶栏菜单、左侧主技法、右侧子页签、关键按钮与设置项的综合 UI 审计
- 修复启动阶段重型技法首开偏慢问题，优化预热优先级与启动探针策略
- 修复在线安装器在替换安装时因运行时资源单一路径 `404` 导致的安装失败问题
- 增强单文件安装器容错：优先识别安装器同目录 runtime 资源，远程下载支持多地址回退
- 完成命盘/事盘/关键设置项关闭后重开可读回的持久化回归验证

## 功能与稳定性修复

- 修复 `关系盘 /modern/relative` 启动阶段首开性能红灯
- 修复 `易与三式 /liureng/gods` 启动阶段首开性能红灯
- 修复综合 UI 审计中 `管理命盘`、`管理事盘`、`新增命盘` 的误报
- 修复易与三式页签卡在 `宿盘` 的回归问题
- 修复六壬页面滚动区域，保证输入区固定、下方结果区可滚动
- 修复主限法表格高度布局回归，底部留白恢复正常
- 修复多处前端重复请求与非必要 loading 造成的卡顿

## 性能优化

- 重排启动预热顺序，优先预热：
  - 六壬
  - 关系盘
  - 星体地图
  - 八字/紫微关键链路
- 调整启动后性能探针等待策略，减少“服务刚起但缓存未热完”导致的假红灯
- 为部分高频链路增加前端静默请求、去重缓存与并发收敛
