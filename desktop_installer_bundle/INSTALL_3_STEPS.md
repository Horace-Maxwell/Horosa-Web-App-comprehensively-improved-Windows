# Horosa Desktop 3-step guide

1. Unzip the package anywhere you want, then open the extracted folder.
2. Double-click `Install_Horosa_Desktop.vbs`.
3. Use the created `Horosa Desktop` shortcut to open Horosa.

Notes:
- Do not open PowerShell scripts manually.
- Updates can be checked from the app menu: `Update -> Check for updates`.
- If Windows asks for network access the first time, allow local access.
- The first install may take a few minutes because the desktop runtime is prepared automatically.
- Your desktop state and update cache are kept in `%LocalAppData%\HorosaDesktop`, so updating the app does not wipe them.
