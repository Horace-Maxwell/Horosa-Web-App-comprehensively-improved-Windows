# 星阙 Windows 桌面版三步安装

## 普通用户下载哪个

- 请到 GitHub Release 下载 `Horosa-Setup-<version>.exe`
- 不要手动下载 `latest.yml` 或 `.blockmap`

## 三步完成安装

1. 下载 `Horosa-Setup-<version>.exe`
2. 双击安装器，按中文向导完成安装
3. 从桌面或开始菜单打开 `星阙`

## 补充说明

- 安装包已经包含 Electron、Java、Python 和前后端资源
- 安装完成后可在本地**离线运行**
- 只有“检查更新”需要网络
- 默认安装到 `%LocalAppData%\\Programs\\Horosa`
- 用户数据保存在 `%LocalAppData%\\HorosaDesktop`
- 如果机器上已安装星阙，安装器会进入维护页，可选择 `替换 / 修复 / 取消`
- 更新可以直接在应用内完成，升级默认保留用户数据
- 安装完成后会自动生成 `星阙` 的桌面快捷方式和开始菜单入口
- 桌面和开始菜单快捷方式由安装器原生创建，名称统一为 `星阙`
